# iso-3166-2-locale
ISO-3166-2 multilingual translation API. Auto-updated from https://github.com/dr5hn/countries-states-cities-database .

## Use

```txt
https://locale.ravelloh.com  // get languages
```

```txt
https://locale.ravelloh.com/zh-CN/  // get countries
```

```txt
https://locale.ravelloh.com/zh-CN/CN/  // get regions
```

```txt
https://locale.ravelloh.com/zh-CN/CN/SD/ // get cities
```
